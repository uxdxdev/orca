# Orca 

![Orca](orca-snapshot.png)

Orca is a CLI tool for downloading your Reddit data;
- Upvoted
- Saved
- Submissions
- Comments

## Help

```terminal
npx @uxdx/orca -h
```

## Usage

```terminal
npx @uxdx/orca --access-token=70162531-eWBggyup_FAKE_Usdf1cz7u-G9pM_dhrVf3g
```

### Install globally

```terminal
npm i @uxdx/orca -g
orca -h
```
