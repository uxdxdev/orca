# Orca 

![Orca](orca.png)

Download your Reddit data:
- Upvoted
- Saved
- Submissions
- Comments

## Help

```terminal
npx @mortond/orca -h
```

## Usage

```terminal
npx @mortond/orca --access-token=70162531-eWBggyup_FAKE_Usdf1cz7u-G9pM_dhrVf3g
```

### Install globally

```terminal
npm i @mortond/orca -g
orca -h
```
